    第2章 变量和基本类型

    基本内置类型

          包括算术类型和空类型，其中算术类型又包括了整数型（包括字符和布尔型在内）、浮点数；空类型呢，又不对应具体的值。
    注意：如何选择合适的类型呢？
  • 当知道数值一定不为负数时，选用无符号类型（无符号类型范围是>=0的）
  • 算术表达式中，不使用short和bool，chart在一些机器上时有符号的，在一些机器上是无符号的
  • 浮点数选用double而非float，因为float通常精度不够而且双精度浮点数和单精度浮点数的计算代价相差无几

extern关键字

      是声明（declaration）的意思，用来区分语句是用来申明的，还是定义的，加上extern关键字后，才能在别处使用，注意，此时不要显示的给出初始化

      变量能且只能被定义一次，但是可以被多次声明

如果在多个文件中使用同一个变量，就必须将声明和定义分离。

两者区别：
1.规定了变量的类型和名字，但是因为不需要初始化，所以无需申请存储空间，但是定义需要申请存储空间。
2.变量的定义必须出现在且只能出现在一个文件中，而其他用到该变量的文件必须对其进行声明，但是绝对不能重复定义

引用：并非是对象，只是作为一个已经存在的对象所起的另外一个名字，所以他是没有实际地址的
所以要记住一点：引用及别名
大名是你，小名还是你，不管怎么称呼，叫的都是你
那么这里需要注意的是，它必须被初始化，否则会被报错，要不然谁知道你引用的是什么呀

那么什么是引用呢？
使用引用的时候，程序把引用和它的初始值绑定在一起，而不是将初始值拷贝给引用。那么一旦绑定，这个引用就无法再绑定到另一个对象，这也是必须初始化的原因。（指针可以指向多个对象哦）
